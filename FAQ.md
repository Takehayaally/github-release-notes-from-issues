# FAQ

## Does this connect to my GitHub account?

No. This MVP reads an exported issues JSON file. It does not require a GitHub token.

## Can it use my own labels?

Yes. Edit `SECTION_RULES` in `release_notes_from_issues.py`.

## What is the paid bundle for?

The paid bundle is for teams that want workflow templates, label presets, and a faster setup guide.

## Refund boundary

Refunds are reasonable when the included examples do not run as described. Custom release process design is not included.
