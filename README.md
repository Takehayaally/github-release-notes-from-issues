# GitHub Issues to Release Notes

Generate release notes from an exported GitHub issues JSON file. It groups closed issues into Added, Fixed, Docs, and Other sections based on labels.

By TinyOps Tools. Support: q749381667@gmail.com.

## Usage

```powershell
python products/github-release-notes-from-issues/release_notes_from_issues.py --issues products/github-release-notes-from-issues/examples/issues.json --version 0.1.0 --out products/github-release-notes-from-issues/examples/RELEASE_NOTES.md
```
